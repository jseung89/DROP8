# DROP 8 소스 전용 버전

DROP 8은 최대 8명이 방 코드로 접속하는 2D 탑다운 세미 배틀로얄 MVP입니다. Phaser가 화면을 그리고 Colyseus 서버가 이동, 총알, 피격, 파밍, AI, 자기장과 승패를 최종 판정합니다.

이 배포본은 Windows Smart App Control의 다운로드 파일 확장자 차단을 줄이기 위해 **TypeScript 소스만 포함**합니다. 배치 파일, PowerShell, JavaScript 빌드 결과물, 실행 파일, `node_modules`는 ZIP에 포함하지 않습니다.

## 준비

1. 공식 Node.js 24 LTS를 설치합니다.
2. ZIP을 압축 해제합니다.
3. VS Code에서 압축 해제한 `DROP8_SOURCE` 폴더를 엽니다.
4. VS Code 상단 메뉴에서 `터미널 → 새 터미널`을 선택합니다.

VS Code는 필수는 아닙니다. Windows Terminal이나 명령 프롬프트에서 프로젝트 폴더로 이동한 뒤 같은 npm 명령을 실행해도 됩니다.

## 설치 및 개발 실행

```text
npm install
npm run dev
```

터미널에 다음 주소가 표시됩니다.

```text
http://localhost:2567
```

브라우저에서 주소를 열고 방을 만들거나 방 코드로 참가합니다. 서버 종료는 터미널에서 `Ctrl+C`입니다.

## 같은 와이파이에서 친구와 플레이

호스트 PC에서 실행합니다.

```text
npm run dev:lan
```

터미널에 다음과 비슷한 주소가 표시됩니다.

```text
내 접속 주소: http://localhost:2567
친구 접속 주소: http://192.168.0.15:2567
```

1. 친구와 호스트 PC를 같은 와이파이에 연결합니다.
2. Windows 방화벽 알림이 나타나면 **개인 네트워크만 허용**합니다.
3. 친구는 표시된 LAN 주소를 브라우저에 입력합니다.
4. 호스트가 만든 6자리 방 코드로 참가합니다.

주소가 표시되지 않으면 Windows에서 `ipconfig`를 실행해 Wi-Fi의 IPv4 주소를 확인하고 `http://IPv4주소:2567`로 접속합니다. 방화벽을 자동 수정하는 스크립트는 포함하지 않습니다.

- `T`: 빠른 채팅

## 알려진 제한

- 그래픽과 맵은 도형 기반 프로토타입입니다.
- AI 경로 탐색과 엄폐 판단은 단순화되어 있습니다.
- 회복·인벤토리 UI는 최소 기능 중심입니다.
- 음성 채팅, 팀전, 총기 부착물, 모바일 조작은 없습니다.
- 수류탄 요리하기, 화염 확산과 AI 직접 투척은 아직 없습니다.
- Phaser가 포함되어 클라이언트 프로덕션 번들이 큽니다.
- 실제 인터넷 환경과 다양한 공유기에서의 장시간 테스트는 추가로 필요합니다.

## 문제 해결

### `npm` 명령을 찾을 수 없음

Node.js 설치 후 VS Code와 터미널을 완전히 닫았다가 다시 엽니다.

### 2567 포트가 사용 중

기존 DROP 8 터미널을 `Ctrl+C`로 종료합니다. 다른 프로그램이 사용 중이라면 `.env.example`을 참고해 다른 포트로 실행할 수 있습니다.

### 친구가 접속하지 못함

- 같은 와이파이인지 확인
- 회사·교육기관 Wi-Fi의 기기 간 통신 차단 여부 확인
- Windows 방화벽에서 Node.js의 개인 네트워크 허용 확인
- VPN 종료 후 재시도

## Refactor 013: 방 시야, 8번 부두와 수영

- 호스트는 로비에서 `작은 맵`, `큰 맵`, `8번 부두`를 선택할 수 있습니다.
- 8번 부두는 7168×7168 규모이며 중앙의 굽은 강, 다리 3개, 여울 2개, 상륙 구간 22개를 포함합니다.
- 대형 건물은 `roomIndex`와 문·창문 포털로 방을 구분합니다. 벽 너머 방의 플레이어·아이템·이름표·미니맵 표시는 숨겨집니다.
- 기존 맵 중앙 건물에도 명시적인 전리품 앵커가 추가됩니다.
- 깊은 물에서는 수영 속도 165px/s가 적용됩니다. 수영 중에는 총기, 근접 공격, 재장전, 투척, 회복, 습득, 오토바이 탑승, 창문 넘기를 사용할 수 없습니다.
- 무기 선택과 채팅은 수영 중에도 가능하며 상륙 후 선택한 무기가 그대로 복원됩니다.
- 파편탄은 물에서도 폭발하고, 연막탄은 깊은 물에서 수증기로 사라지며, 화염탄은 깊은 물에서 소화됩니다.
- 오토바이는 차량용 다리와 관리 다리만 통과하며 깊은 물과 보행자 다리·여울에는 진입할 수 없습니다.
