## 2. 수풀 인지

- [ ] AI 시야 밖에서 수풀 안에 먼저 숨은 플레이어를 AI가 원거리에서 발견하지 못한다.
- [ ] AI가 플레이어를 본 뒤 플레이어가 수풀에 들어가면 잠깐만 마지막 이동을 추정한다.
- [ ] 이후 AI 조준이 수풀 안 플레이어의 실시간 좌표를 계속 따라가지 않는다.
- [ ] AI가 마지막 목격 위치까지 이동한 뒤 주변을 수색한다.
- [ ] 수풀 안에서 총을 쏘면 짧게 노출되거나 소리 조사 상태로 전환된다.
- [ ] 단발 후 위치를 바꾸면 AI가 이전 발사 위치를 조사한다.
- [ ] AI가 수풀 바로 옆까지 접근하면 플레이어를 발견할 수 있다.
- [ ] 수풀 속 AI의 말풍선이 캐릭터 위치를 부당하게 노출하지 않는다.

## 3. 벽·건물·방

- [ ] 완전히 막힌 외벽 너머 플레이어를 AI가 발견하지 못한다.
- [ ] 다른 방 플레이어에게 AI가 즉시 발사하지 않는다.
- [ ] 문이나 창문을 통해 실제 시야가 연결됐을 때만 발견한다.
- [ ] 문 여러 개가 일렬로 놓여 있어도 플레이어 렌더 규칙보다 과도하게 투시하지 않는다.
- [ ] 목표가 벽 뒤로 숨으면 AI가 현재 좌표가 아닌 마지막 목격 위치로 이동한다.
- [ ] AI가 목표를 찾지 못하면 3~5초 안에 추적을 포기한다.

## 4. 시야각과 반응시간

- [ ] 평상시 AI 뒤쪽으로 접근하면 즉시 발견하지 않는다.
- [ ] 측면 발견은 정면 발견보다 반응이 느리다.
- [ ] 뒤에서 소리가 나면 먼저 회전한 뒤 확인한다.
- [ ] AI가 적 발견 즉시 같은 프레임에 발사하지 않는다.
- [ ] 목표 방향으로 몸과 무기를 돌린 뒤 발사한다.
- [ ] 난이도 Hard도 시야각·반응시간을 완전히 무시하지 않는다.

## 5. 청각

- [ ] 권총 소리는 가까운 AI만 조사한다.
- [ ] 소총·저격총·바주카포는 더 먼 AI도 반응한다.
- [ ] 은화살은 일반 총기보다 짧은 거리에서 인지된다.
- [ ] 달리기는 걷기보다 더 멀리 들린다.
- [ ] 움직이지 않는 플레이어는 발소리를 만들지 않는다.
- [ ] 오토바이가 접근하면 AI가 차량 소리 방향을 조사한다.
- [ ] 소리만 들은 AI가 플레이어 몸 중심으로 바로 사격하지 않는다.
- [ ] 조사 위치에 실제 소리 위치와 적당한 오차가 있다.

## 6. 조준과 샷발

- [ ] AI 첫 탄이 항상 캐릭터 중심에 맞지 않는다.
- [ ] 근거리보다 중·장거리 오차가 크다.
- [ ] 이동 중 AI 명중률이 정지 상태보다 낮다.
- [ ] 급회전 직후 바로 완벽하게 맞히지 못한다.
- [ ] 조준 오차가 매 프레임 심하게 떨리는 형태가 아니다.
- [ ] 일정 시간 조준하면 일부 안정화되는 느낌이 있다.
- [ ] AI 성격과 난이도에 따라 실력 차이가 보인다.

## 7. 총기별 행동

- [ ] 소총 AI가 3~6발 점사 후 잠깐 쉰다.
- [ ] SMG AI가 근거리에서 5~10발 점사한다.
- [ ] 샷건 AI가 유효거리 밖에서 낭비 사격하지 않는다.
- [ ] 샷건 한 발 후 이동 또는 거리 조절을 한다.
- [ ] 저격총 AI가 조준시간 없이 즉시 발사하지 않는다.
- [ ] 바주카 AI가 최소 안전거리 안에서 발사를 보류한다.
- [ ] 화염방사기 AI가 사거리 밖 벽에 계속 분사하지 않는다.
- [ ] 은화살 AI가 현재 산탄·연사·유도 보정을 유지한다.

## 8. 재장전·회복·후퇴

- [ ] 빈 탄창 AI가 계속 발사 입력만 반복하지 않는다.
- [ ] 탄창이 적고 거리가 충분하면 미리 재장전할 수 있다.
- [ ] 재장전 중 적이 가까우면 후퇴하거나 엄폐를 찾는다.
- [ ] `재장전!` 말풍선이 실제 재장전 시작 시에만 나온다.
- [ ] 체력 40% 이하에서 후퇴와 엄폐 비중이 증가한다.
- [ ] 체력 20% 이하에서 생존 행동이 우선된다.
- [ ] 회복 아이템이 있으면 안전한 위치에서 사용한다.
- [ ] `잠깐 빠질게`, `회복할 시간이 필요해` 대사와 행동이 일치한다.

## 9. 건물 체류와 탈출

- [ ] AI가 아이템 없는 방에 계속 머물지 않는다.
- [ ] 같은 방에 목표 없이 약 2~3초 이상 정지하지 않는다.
- [ ] 같은 건물에 약 10~15초 이상 무목적으로 머물지 않는다.
- [ ] 탐색 종료 시 `여긴 다 봤어`, `밖으로 나가자` 대사가 나온다.
- [ ] 대사 후 실제 출구 방향으로 이동한다.
- [ ] 문 중앙에서 멈추지 않고 문 바깥 좌표까지 이동한다.
- [ ] 문이 막히면 창문 또는 다른 출구를 선택한다.
- [ ] 기존 창문 vault가 정상 작동한다.

## 10. 갇힘 복구

- [ ] 벽이나 가구에 막히면 먼저 좌우 우회를 시도한다.
- [ ] 같은 출구를 반복 실패하면 다른 출구를 고른다.
- [ ] 두 번 이상 막히면 건물 출구를 우선 찾는다.
- [ ] `길이 막혔어, 돌아가` 대사 후 실제 목표가 바뀐다.
- [ ] 벽을 통과하는 순간이동이 발생하지 않는다.
- [ ] 다른 플레이어 바로 옆으로 강제 이동하지 않는다.
- [ ] 복구 후 다시 정상 순찰·파밍·전투로 복귀한다.

## 11. AI 말풍선과 현장 채팅

- [ ] AI 대사가 기존 플레이어 머리 위 말풍선 형태로 표시된다.
- [ ] AI 말풍선은 약 4초 뒤 사라진다.
- [ ] 가까운 AI 대사만 왼쪽 로그에 `[AI 음성]`으로 표시된다.
- [ ] AI 대사와 플레이어 채팅의 색상이 구분된다.
- [ ] 먼 AI 대사가 전체 시스템 메시지로 나오지 않는다.
- [ ] 벽 너머 AI의 머리 위치에 말풍선이 표시되지 않는다.
- [ ] 수풀 속 AI 말풍선이 위치를 공개하지 않는다.
- [ ] 같은 문장이 30초 안에 반복적으로 쏟아지지 않는다.
- [ ] 전투 중 평상시 잡담이 과도하게 나오지 않는다.
- [ ] 사망한 AI는 말하지 않는다.

## 12. 호출·응답

- [ ] 가까운 AI가 불확실한 움직임·목표 상실·출구·갇힘 대사에 짧게 응답할 수 있다.
- [ ] 응답 AI가 플레이어의 정확한 현재 좌표를 받지 않는다.
- [ ] 응답 AI는 오차가 포함된 조사 위치만 얻는다.
- [ ] 모든 AI에게 전역으로 정보가 전송되지 않는다.
- [ ] 응답 대사가 6초 이내에 반복되지 않는다.

- [ ] 여러 클라이언트에서 같은 AI 행동과 대사가 일관된다.
- [ ] AI 대사가 네트워크 로그를 과도하게 증가시키지 않는다.
- [ ] 경기 종료 후 AI 기억·성격·대화 타이머가 정리된다.
- [ ] 재경기에서 이전 목표·대사·수색 상태가 남지 않는다.
