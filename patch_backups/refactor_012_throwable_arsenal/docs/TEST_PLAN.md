# 테스트 계획

## 자동 테스트

`npm run check`로 다음을 실행합니다.

- 공유 규칙 단위 테스트
- 방 코드 및 채팅 정제
- 방 생성과 사용자 2명 동기화
- 실제 사용자 1명과 AI 충원
- 실제 사용자 8명 동시 참가
- 서버 권위 이동 입력
- TypeScript 타입 검사
- ESLint
- 클라이언트·서버 빌드

## 수동 테스트

1. 브라우저 탭 2개에서 방 생성과 참가
2. 로비 채팅과 준비 상태 확인
3. AI 충원 후 경기 시작
4. 두 사용자의 비행기 낙하 확인
5. 아이템 획득과 총기 전환
6. 피격, 사망, 관전, 우승
7. 리매치 후 이전 총알·아이템·체력 초기화
8. `npm run dev:lan`으로 다른 PC 접속
