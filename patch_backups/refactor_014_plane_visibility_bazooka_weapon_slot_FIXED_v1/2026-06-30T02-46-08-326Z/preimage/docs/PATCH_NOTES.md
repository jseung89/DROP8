- 창문 넘기 진입점은 인간 클라이언트 메시지 전용이었고 `p.ai`를 명시적으로 거부했습니다.

- 화염·수류탄 회피는 목적지만 바꾸고 경로를 비워 건물 내부에서 직선 충돌을 반복할 수 있었습니다.

- 여섯 지역에 고유 바닥·벽·건물 색상과 지역 설명을 적용했습니다.

- 장식물은 이번 버전에서 시각 요소이며 충돌 장애물로 사용하지 않습니다.
- 필드 채팅 고정 표시, 재장전 연출, 건물 끼임 추가 안정화는 Refactor 004C 범위입니다.

## Refactor 004C UX Hotfix — 채팅 입력, 닉네임, 머리 위 말풍선

- Enter 입력을 DOM 채팅 input 한 곳에서 처리해 입력창이 전송 전에 닫히던 충돌을 수정했습니다.
- 채팅 input 이벤트가 Phaser 전역 키 처리로 전파되지 않도록 하여 Space 공백 입력과 한글 IME 조합을 정상화했습니다.

- 서버가 playerId, 서버 권위 닉네임, 전송 시각을 포함한 메시지를 보낸 사람과 수신자에게 동일하게 전달합니다.
- 살아 있는 플레이어와 AI 위에 재사용 가능한 닉네임 오버레이를 표시하고 내 캐릭터에는 `(나)`를 붙입니다.

- 부쉬로 보이지 않는 플레이어는 닉네임과 말풍선도 함께 숨깁니다.

알려진 제한:

- 음성 채팅, 팀 전용 채팅, AI 자동 채팅은 제공하지 않습니다.
- 말풍선은 최대 두 줄의 텍스트만 표시하며 커스텀 이모지는 지원하지 않습니다.
