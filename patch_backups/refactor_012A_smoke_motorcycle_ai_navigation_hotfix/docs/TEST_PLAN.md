## Refactor 012 수동 QA 체크리스트

1. 4번으로 투척 슬롯을 선택하고 Q로 자동 회복한다.
2. 채팅 입력 중 Q·1~4·E·R·마우스 투척이 실행되지 않는다.
3. 짧은 클릭과 0.85초 최대 충전의 거리가 다르다.
4. 가이드가 벽 충돌·튕김·착지점·효과 반경을 실제 서버 결과와 동일하게 표시한다.
5. 열린 문으로 실내외 투척이 가능하다.
6. 창문 중앙은 통과하고 낮거나 높은 궤적은 창틀에 튕긴다.
7. 파편탄 피해 구간, 벽 차폐, 자기 피해, 차량 피해와 처치 귀속을 확인한다.
8. 연막 밖↔안 시야가 차단되고 같은 연막 80px 이내 실루엣만 보인다.
9. 연막 중 닉네임·체력바·말풍선이 숨고 총알·소리·폭발은 통과한다.
10. 화염은 초기 8, 초당 12이며 밖으로 나오면 즉시 멈추고 중첩되지 않는다.
11. AI가 연막 너머를 사격하지 않고 화염·임박한 파편탄에서 이탈한다.
12. 다른 종류 획득 시 기존 투척 스택이 바닥에 남고 같은 종류는 최대 3개까지 합쳐진다.
13. 수량 0에서 직전 무기로 자동 복귀한다.
14. 두 브라우저 또는 PC에서 투척 위치·효과·피해·사운드가 동기화된다.
